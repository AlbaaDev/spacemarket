# SpaceMarket
SpaceMarket is a CRM project currently in development using Angular 20, Java 24 Spring 3.5.3, and PostgreSQL.

---

## Installation

1. Go to the root project folder (`SpaceMarket`).

2. Open a terminal and run both the backend and frontend with this command:

```bash
node start.js
```
